# Docklift Test Projects

This folder contains simple test projects to test Docklift deployments.

## Projects

| Folder | Type | Port |
|--------|------|------|
| `python-test` | Python Flask API | 5000 |
| `nextjs-test` | Next.js Web App | 3000 |
| `server-test` | Node.js Express Server | 4000 |

## Testing

1. Zip any folder (e.g., `python-test/`)
2. Upload to Docklift
3. Click Deploy
4. Access at assigned port
